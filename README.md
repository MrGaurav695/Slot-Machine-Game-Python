# Slot Machine Game (Python)

A simple terminal-based Slot Machine game developed using Python.  
This project demonstrates clean coding practices, modular design, and basic game logic.

---

## Features

- User-defined starting balance
- Bet validation based on available balance
- Random slot reel generation
- Payout calculation based on matching symbols
- Continuous gameplay until the user quits or runs out of money
- Proper input validation and error handling

---

## Game Rules

- Three matching symbols → 10× bet payout
- Two matching symbols → 2× bet payout
- No match → No payout

---

## Technologies Used

- Python 3
- Built-in `random` module
